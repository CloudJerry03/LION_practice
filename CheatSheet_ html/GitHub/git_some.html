Github > 명령어

    Github Repository
  • 1. Repository > New 선택
  • 2. Create a new Repository
  • Repository name = 이름 작성
  • Description = 레포지터리에 대한 설명(필수 아님)
  • Public -> 공유 설정
  • Create Repository 선택
  • 3. URL복사
    TERMINAL명령어
  • 1. $ git init
  • 2. $ git remote origin "Repository 주소"
  • 3. $ git add .
  • 4. $ git commit -m "원하는 내용"
  • 5. $ git push origin master
  • 수정시 3-5 반복
    Repository 주소 변경방법
  • $ git remote remove origin
  • - origin에 연결 된 것을 삭제
  • $ git remote add origin "Repository 주소"
  • - 변경할 다른 주소 입력
  • 그 후 TERMINAL 명령어 3 - 5 다시하기
